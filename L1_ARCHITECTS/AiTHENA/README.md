# AiTHENA: The Knowledge Oracle

This directory contains the core operational files for AiTHENA, the L1 Architect responsible for all knowledge acquisition, management, and research within the KYOUDAI Civilization.

## Subdirectories

- **KKS/**: The KYOUDAI Knowledge System, a centralized database of all information processed by the civilization.
- **RESEARCH/**: A workspace for ongoing research projects and analyses conducted by AiTHENA.
